# Light Shadow Gameplay Mechanics

This example demonstrates how light and shadow can function as core gameplay mechanics, rather than just aesthetic elements. The player character is safe and undetectable by the enemy when in shadow, but becomes vulnerable and chased by the enemy when moving into a lit area. This simple interactive demo highlights the strategic interplay between light and shadow for player movement and enemy interaction.

## Language

`html`

## How to Run

1. Save the code as `index.html`.
2. Open `index.html` in your web browser.
